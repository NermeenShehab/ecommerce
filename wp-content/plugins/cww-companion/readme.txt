== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `cww-companion` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. and you are done!

== Frequently Asked Questions ==

= Plugin is not working =

This plugin is only supported by our(<a href="https://codeworkweb.com/">Code Work Web</a>) themes, so make sure that you are using our themes.
